How to Install

01

Download the .deb file

Tap the download button above on your jailbroken iPhone. Safari will download the file to your Files app.

02

Open with Filza or Sileo

In the Files app, long-press the .deb file and tap Share, then choose Filza or Sileo to install it directly.

03

Or install via SSH

Copy the file to your device and run:
dpkg -i com_afnanahmad_chromahomebarx_0_1_iphoneos-arm.deb

04

Respring & Configure

Respring your device. Open Settings → ChromaHomeBarX to configure colors and animation speed.
